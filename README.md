## 트러블 슈팅 웹 서비스 Front-End

### TESTSERVER
https://www.notworking.kr/

### 사용 기술

> Language : TypeScript <br/>
> Framework : NextJS <br/>
> Style : Styled-Components<br/>
> State Management : Client-side : Recoil , Server-side : React-query <br/>
> Tools : LightHouse

### LIGHT HOUSE SCORE

> mobile+LowLevel : 50
> mobile=MidLevel : 80
> pc : 95

### Getting Started

RUN DEV : run the development server:

```bash
npm run dev
# or
yarn dev
```

RUN BUILD :

```bash
npm run build
# or
yarn run build
```
