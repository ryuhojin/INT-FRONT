## 트러블 슈팅 웹 서비스 Front-End

### 사용 기술

> Language : TypeScript <br/>
> Framework : NextJS <br/>
> Style : TailwindCSS <br/>
> State Management : Redux + Redux Thunk <br/>
> Tools : LightHouse

### Getting Started

RUN DEV : run the development server:

```bash
npm run dev
# or
yarn dev
```

RUN BUILD :

```bash
npm run build
# or
yarn run build
```
